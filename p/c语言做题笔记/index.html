从字符串中删除特定字符（蒋玉茹审核）

题目描述

输入字符串s和字符c，要求去掉s中所有的c字符，并输出结果。

输入格式

测试数据有多组，每组输入字符串s和字符c。

输出

对于每组输入,输出去除c字符后的结果。

#include<stdio.h>
int main()
{
    char s[100]={0};
    char c;
    printf("please input s:");
    scanf("%s",&s);
    while (getchar() != '\n');       //此行代码特殊
    printf("please input c:");
    scanf("%c",&c);
    printf("Output:\n");
    for (int i = 0 ; s[i] != 0 ; i++)
    {
        if (s[i] != c){
            printf("%c",s[i]);
        }
    }
}

若无特殊行，输入s的字符串后会跳过对c的输入，导致结果错误

以下内容来自AI回答

具体错误

  • scanf("%s", s); 读取字符串时，会在字符串末尾添加一个 \0 字符，并且会忽略输入中的空白字符（如空格、换行符等）。
  • 当你输入字符串后按下回车键时，回车符（\n）会被留在输入缓冲区中。
  • scanf("%c", &c); 会立即读取这个留在缓冲区中的回车符，而不是等待你输入新的字符。

解决方法

在读取字符 c 之前，清除输入缓冲区中的剩余字符（包括回车符）。可以使用 getchar() 来清除缓冲区中的字符。

工作原理

while (getchar() != '\n'); 这段代码的作用是清除输入缓冲区中的剩余字符，直到遇到换行符 \n。下面是详细的解释：

getchar() 函数

  • getchar() 是一个标准输入函数，用于从标准输入（通常是键盘）读取一个字符。
  • 它返回读取的字符的 ASCII 值，如果遇到文件结束符（EOF），则返回 EOF。

while (getchar() != '\n'); 的工作原理

  1. 读取字符：getchar() 每次从输入缓冲区中读取一个字符。
  2. 循环条件：while (getchar() != '\n'); 中的条件是 getchar() != '\n'，即只要读取的字符不是换行符 \n，循环就会继续。
  3. 清除缓冲区：在循环中，getchar() 会不断读取并丢弃输入缓冲区中的字符，直到遇到换行符 \n。
  4. 结束循环：当 getchar() 读取到换行符 \n 时，条件 getchar() != '\n' 为假，循环结束。

具体步骤

假设用户输入了以下内容并按下回车键：

hello world
  1. 输入缓冲区内容：
    • 输入缓冲区中会有字符 h, e, l, l, o, ``, w, o, r, l, d, \n。
  2. 执行 while (getchar() != '\n');：
    • getchar() 读取字符 h，条件为真，继续循环。
    • getchar() 读取字符 e，条件为真，继续循环。
    • getchar() 读取字符 l，条件为真，继续循环。
    • getchar() 读取字符 l，条件为真，继续循环。
    • getchar() 读取字符 o，条件为真，继续循环。
    • getchar() 读取字符 ``，条件为真，继续循环。
    • getchar() 读取字符 w，条件为真，继续循环。
    • getchar() 读取字符 o，条件为真，继续循环。
    • getchar() 读取字符 r，条件为真，继续循环。
    • getchar() 读取字符 l，条件为真，继续循环。
    • getchar() 读取字符 d，条件为真，继续循环。
    • getchar() 读取字符 \n，条件为假，循环结束。
  3. 结果：
    • 输入缓冲区中的所有字符（包括换行符 \n）都被读取并丢弃，缓冲区被清空。

总结

while (getchar() != '\n'); 这段代码通过不断读取并丢弃输入缓冲区中的字符，直到遇到换行符 \n，从而清空输入缓冲区。这样可以确保后续的输入操作（如 scanf("%c", &c);）不会受到之前输入的干扰。

提出的问题

所以实际上我对s数组输入每一个字符时while语句都会执行一遍吗

回答

不，while (getchar() != '\n'); 这段代码并不会在你输入 s 数组的每一个字符时都执行一遍。它的执行时机是在你输入完字符串并按下回车键之后。

详细解释

  1. 输入 s 数组：
    • 当你使用 scanf("%s", s); 读取字符串时，scanf 会从标准输入中读取字符，直到遇到空白字符（如空格、制表符、换行符等）。
    • 假设你输入了 hello world 并按下回车键，scanf("%s", s); 会读取 hello，并将 world 和换行符 \n 留在输入缓冲区中。
  2. 输入缓冲区状态：
    • 输入缓冲区中会有字符 world 和换行符 \n。
  3. 执行 while (getchar() != '\n');：
    • 这段代码会在 scanf("%s", s); 之后执行。
    • 它会读取并丢弃输入缓冲区中的字符，直到遇到换行符 \n。
    • 具体步骤如下：
      • getchar() 读取字符 w，条件为真，继续循环。
      • getchar() 读取字符 o，条件为真，继续循环。
      • getchar() 读取字符 r，条件为真，继续循环。
      • getchar() 读取字符 l，条件为真，继续循环。
      • getchar() 读取字符 d，条件为真，继续循环。
      • getchar() 读取字符 \n，条件为假，循环结束。
  4. 结果：
    • 输入缓冲区中的所有字符（包括换行符 \n）都被读取并丢弃，缓冲区被清空。

总结

  • while (getchar() != '\n'); 只会在你输入完字符串并按下回车键之后执行一次。
  • 它的作用是清除输入缓冲区中剩余的字符，确保后续的输入操作（如 scanf("%c", &c);）不会受到之前输入的干扰。

补充——可以读取带空格的字符串的方法

gets函数

如果你希望进一步简化代码，可以使用 gets 函数来读取带空格的字符串。不过需要注意的是，gets 函数存在缓冲区溢出的风险，因此在实际开发中不推荐使用。尽管如此，为了回答你的问题，这里还是提供一个使用 gets 的示例。

#include <stdio.h>

int main() {
    char s[100] = {0};
    char c;

    printf("please input s: ");
    gets(s);  // 读取带空格的字符串

    printf("please input c: ");
    scanf(" %c", &c);  // 读取字符

    printf("Output:\n");
    for (int i = 0; s[i] != 0; i++) {
        if (s[i] != c) {
            printf("%c", s[i]);
        }
    }

    return 0;
}
解释
  1. 读取带空格的字符串：
    • gets(s); 读取一行字符串，直到遇到换行符。
    • gets 会自动处理换行符，并将字符串存储到 s 数组中。
  2. 读取字符 c：
    • scanf(" %c", &c); 读取一个字符。
    • 注意 %c 前面的空格，用于跳过空白字符（如换行符）。
  3. 输出结果：
    • 遍历 s 数组，输出不等于 c 的字符。
总结

使用 gets 函数可以非常简单地读取带空格的字符串。然而，由于 gets 存在缓冲区溢出的风险，建议在实际开发中避免使用它。如果你需要更安全的解决方案，建议使用 fgets 或 scanf 的 %[^\n] 格式。

scanf的格式化输入

当然有更简单的方法！你可以使用 scanf 的格式化输入来读取带空格的字符串。scanf 的 %[^\n] 格式可以读取一行字符串，直到遇到换行符。

#include <stdio.h>

int main() {
    char s[100] = {0};
    char c;

    printf("please input s: ");
    scanf(" %[^\n]", s);  // 读取带空格的字符串

    printf("please input c: ");
    scanf(" %c", &c);  // 读取字符

    printf("Output:\n");
    for (int i = 0; s[i] != 0; i++) {
        if (s[i] != c) {
            printf("%c", s[i]);
        }
    }

    return 0;
}
解释
  1. 读取带空格的字符串：
    • scanf(" %[^\n]", s); 使用 %[^\n] 格式读取一行字符串，直到遇到换行符。
    • 注意 %[^\n] 前面的空格，用于跳过空白字符（如换行符）。
  2. 读取字符 c：
    • scanf(" %c", &c); 读取一个字符。
    • 注意 %c 前面的空格，用于跳过空白字符（如换行符）。
  3. 输出结果：
    • 遍历 s 数组，输出不等于 c 的字符。
总结

使用 scanf(" %[^\n]", s); 可以非常方便地读取带空格的字符串，而不需要手动处理换行符或其他复杂的逻辑。这种方法简洁且易于理解。

手动读取字符

当然可以！如果你不想使用其他标准函数（如 fgets 或 strcspn），你可以手动处理输入和换行符。以下是一个不使用其他标准函数的示例代码：

#include <stdio.h>

int main() {
    char s[100] = {0};
    char c;
    int i = 0;

    printf("please input s: ");
    while (i < sizeof(s) - 1) {
        char ch = getchar();
        if (ch == '\n') {
            break;
        }
        s[i++] = ch;
    }
    s[i] = '\0';  // 手动添加字符串终止符

    printf("please input c: ");
    scanf(" %c", &c);  // 注意前面的空格，用于跳过空白字符

    printf("Output:\n");
    for (int j = 0; s[j] != 0; j++) {
        if (s[j] != c) {
            printf("%c", s[j]);
        }
    }

    return 0;
}
解释
  1. 手动读取字符串：
    • 使用 getchar() 逐个读取字符，直到遇到换行符 \n 或达到数组的最大长度。
    • while (i < sizeof(s) - 1) 确保不会超出数组的边界。
    • s[i++] = ch; 将读取的字符存储到数组中。
    • s[i] = '\0'; 手动添加字符串终止符。
  2. 读取字符 c：
    • scanf(" %c", &c); 读取一个字符。
    • 注意 %c 前面的空格，用于跳过空白字符（如换行符）。
  3. 输出结果：
    • 遍历 s 数组，输出不等于 c 的字符。
总结

通过手动读取字符并处理换行符，你可以实现不使用其他标准函数的功能。这种方法虽然稍微复杂一些，但可以完全控制输入的处理过程。
